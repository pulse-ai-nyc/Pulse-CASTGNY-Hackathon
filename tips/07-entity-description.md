# Tip 7: Write a Clear Entity Description

**What it is:** A short, factual summary on the brand's site that clearly states what the brand is, who it's for, and what makes it different — in a format that AI models can easily quote.

**How to measure it:**
- Check the brand's homepage and About page for a concise "what we are" statement
- Test: Can you extract a 1–2 sentence description of the brand from the page?
- Compare to competitors: do they have a clearer "elevator pitch" on their site?

**Example:**
> Good: "Retool is a low-code platform that lets engineering teams build internal tools 10x faster."  
> Bad: A homepage with only a tagline like "Build Better." and no context.

**Why it matters:**
When a model answers "What is X?", it looks for a clear, quotable description. If your site doesn't provide one, the model either invents one (risk of hallucination) or skips you for a competitor whose description is easy to extract.

**What your agent should do:**
- Check if the brand has a clear entity description
- If missing, generate a Wikipedia-style summary draft the brand can publish
- Recommend placement: About page, homepage hero, and meta description

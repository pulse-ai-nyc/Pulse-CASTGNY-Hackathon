# Tip 1: Measure Share of Model (SoM)

**What it is:** The percentage of high-intent queries where the brand appears in the AI-generated answer.

**How to measure it:**
- Run 5–10 high-intent queries across 2+ AI models (ChatGPT, Perplexity, Gemini, Claude, etc.)
- For each query, check if the brand is mentioned in the response
- SoM = (queries where brand appears / total queries) × 100

**Example:**
> You run 5 queries. The brand appears in 2 of them.  
> SoM = 2/5 = **40%**

**Why it matters:**
This is the single most important metric for GEO. If SoM is 0%, the brand is invisible to anyone using AI search. Tracking it over time shows whether your content strategy is working.

**What your agent should do:**
- Compute SoM automatically from live query results
- Display it prominently in the report (e.g. "Your SoM is 40% across 5 queries on 2 models")
- Set a target (e.g. "Goal: 80%")
